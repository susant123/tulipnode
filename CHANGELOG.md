# Changelog

All notable changes to this maintained fork will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-02

### Changed
- **Forked from original tulipnode**: Created maintained fork to provide ongoing support
- **Package name**: Changed from `tulind` to `tulind-maintained`
- **Updated dependencies**: 
  - Updated `nan` to ^2.17.0
  - Updated `bindings` to ^1.5.0
  - Updated `@mapbox/node-pre-gyp` to ^1.0.11
- **Repository URL**: Updated to point to maintained fork
- **Documentation**: Updated README with fork information and attribution
- **Node.js support**: Confirmed compatibility with Node.js 16, 18, 20, and 22

### Added
- This CHANGELOG.md file
- Maintainer information in package.json
- Contributing guidelines in README
- Clear attribution to original author Lewis Van Winkle
- Fork notice in README

### Maintained
- All original functionality from tulipnode v0.8.20
- LGPL-3.0 license (unchanged)
- Full API compatibility with original package
- 100+ technical analysis indicator functions
